# **八、一个关键优化建议（避免后期混乱）**

你需要避免一个坑：

❌ 标签随意增长没有治理

建议你加一个“标签治理规则”：

---

## **✔️ 标签规则**

```text
1. 标签必须归属某一类（scenario / problem / ui / concept）
2. 禁止重复语义标签（如 usability vs user-friendly）
3. scenario 优先于 concept
4. problem 标签优先用于分析
5. ui 标签仅描述界面结构
6. 每条内容最多：
   - scenario ≤ 2
   - problem ≤ 3
   - ui ≤ 5
```
