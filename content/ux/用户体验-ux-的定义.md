摘要：“用户体验”涵盖了终端用户与公司、其服务以及其产品交互的所有方面。

优秀用户体验的首要要求是顺畅且丝毫不差地满足客户的确切需求。其次是产品的简洁与优雅，使其成为人们乐于拥有、乐于使用的产品。真正的用户体验远不止于给客户提供他们口头上说想要的东西，或是提供一份功能清单。为了在公司的产品和服务中实现高质量的用户体验，必须无缝整合多个学科的服务，包括工程、市场营销、图形与工业设计，以及界面设计。

必须将整体的用户体验与用户界面（UI）区分开来，尽管 UI 显然是设计中极其重要的一部分。举个例子，假设有一个电影评论网站。即便用来寻找电影的 UI 设计得完美无瑕，但如果底层数据库仅包含主流制片厂的电影，那么对于一个想要寻找小型独立发行电影信息的用户来说，其 UX 依然是糟糕的。

我们还应该**区分 UX 与可用性（Usability）**：根据[可用性的定义](https://www.nngroup.com/articles/usability-101-introduction-to-usability/)，它是 UI 的一种质量属性，涵盖了系统是否易学、使用是否高效、是否令人愉悦等维度。同样，这非常重要，但整体用户体验依然是一个更为广泛的概念。
