ARIA 编写规范指南

## ARIA 编写规范指南 (APG) 首页

学习使用可访问富互联网应用程序（ARIA）规范中定义的可访问性语义，来创建可访问的网页体验。本指南介绍了如何将可访问性语义应用到常见的设计模式和微件（widgets）中。它提供了设计模式和功能示例，并辅以针对基础实践的深入指导。

[查看设计模式](https://www.w3.org/WAI/ARIA/apg/patterns/)

![笔记本电脑屏幕上显示着一个可访问性图标，并弹出一个复选标记。](https://www.w3.org/WAI/content-images/wai-aria-practices/images/index-1.svg)

## APG 资源

帮助您构建可访问网页的基石

### 设计模式与示例

学习如何利用 ARIA 角色（roles）、状态（states）和属性（properties）以及通过实现键盘支持来构建可访问的网页组件和微件。本指南通过功能示例展示了实现每种模式的一种或多种方法。

[了解更多](https://www.w3.org/WAI/ARIA/apg/patterns/)

![各种各样的微件。](https://www.w3.org/WAI/content-images/wai-aria-practices/images/index-2.svg)

### 使用 ARIA 地标

学习如何使用 HTML 分区元素和 ARIA 地标角色，从而让辅助技术用户能够轻松理解页面布局的含义。

[了解更多](https://www.w3.org/WAI/ARIA/apg/practices/landmark-regions/)

![一份文档散开成多个区块。](https://www.w3.org/WAI/content-images/wai-aria-practices/images/index-3.svg)

### 提供可访问的名称和描述

为元素提供可访问的名称，并在合适时提供可访问的描述，是开发者在构建可访问网页体验时最重要的职责之一。

[了解更多](https://www.w3.org/WAI/ARIA/apg/practices/names-and-descriptions/)

![指示器深入探入文档内部。](https://www.w3.org/WAI/content-images/wai-aria-practices/images/index-4.svg)

## 参与其中

APG 工作组（APG Task Force）依赖广泛的社区代表性和参与度，来持续提升 APG 的实用性和质量。您可以通过多种方式参与进来，帮助推动可访问体验的发展。

![显示三个节点相连的图标。](https://www.w3.org/WAI/content-images/wai-aria-practices/images/index-6.svg)

### 加入工作组

要加入 APG 工作组，个人需要首先加入 W3C ARIA 工作组（W3C ARIA Working Group）。我们期望参与者能够积极为工作组的工作做出贡献。

[了解更多关于工作组的工作以及如何加入的信息](https://www.w3.org/WAI/ARIA/task-forces/practices/)

![显示两个人形合力搬运重物的图标。](https://www.w3.org/WAI/content-images/wai-aria-practices/images/index-7.svg)

### 通过 GitHub 做出贡献

许多宝贵的贡献来自于在我们的 GitHub 仓库中发现或提出感兴趣的议题（issues），然后通过 GitHub 拉取请求（pull request）提交修改建议的成员。如果您选择这条途径，请首先学习我们的仓库贡献指南以及维护代码质量的规范。

[在 GitHub 仓库中查看 ReadMe](https://github.com/w3c/aria-practices)

![一封电子邮件图标上方出现通知铃铛图标。](https://www.w3.org/WAI/content-images/wai-aria-practices/images/index-8.svg)

### 邮件列表

APG 工作组使用公开的 aria-practices 邮件列表进行邮件讨论。会议通知、议程以及会议纪要的链接都会发送到该邮件列表。虽然 GitHub 议题（issues）是讨论 APG 内容的首选场所，但对于更倾向于通过电子邮件与 APG 工作组进行沟通的人，邮件列表同样适用。

[查看 aria-practices 邮件列表存档](https://lists.w3.org/Archives/Public/public-aria-practices/)
